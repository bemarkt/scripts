## 使用说明

部分网站模块因源网站套了 Cloudflare 原因，可能需要使用到：[CloudflareBypassForScraping fork版](https://github.com/bemarkt/CloudflareBypassForScraping)。

配置好 CloudflareBypassForScraping Server 后，在模块填写`bypass_url`，如：`http://foo.bar:8000`
